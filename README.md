# Descripción del proyecto

Este proyecto consiste en el diseño y ejecución de consultas SQL sobre la base de datos Sakila gestionada con DBeaver.
El objetivo es demostrar el manejo completo de SQL: desde consultas simples sobre una única tabla hasta relaciones entre tablas, subconsultas y vistas.

# Estructura del repositorio
